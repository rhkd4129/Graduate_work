로컬에서 브랜치삭제 햇을때 온라인으로도 삭제하려면
git push --delete <remote> <branch>    remote는 일반적으로 origin   branch는 삭제할 브렌치


로컹에서 브랜치 만들고 푸쉬하는 방법은
git push <remote> <branch>

git fetch origin으로 브랜치 최신 정보 가져오기
git checkout a 브랜치 가져오기
git push --set-upstream origin image_upload


git branch -r 원격 브랜치 조회
git checkout -t [원격 브랜치명]
